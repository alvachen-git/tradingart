一、碳酸锂核心定位：从“工业味精”到“白色石油” 
基本属性： 碳酸锂是锂电池正极材料的核心原料，被称为“白色石油” 。 
碳酸锂产业链地位： 
上游： 原料主要为锂辉石（澳洲为主）、锂云母（江西为主）和盐湖卤水（南美及青海、西藏） 。
中游： 冶炼加工，中国是全球最大的碳酸锂生产国（产量占全球63%） 。 
下游： 锂电池占绝对主导地位（93%），其中正极材料占81%，传统工业（玻璃陶瓷等）仅占7% 。

二、碳酸锂供给端逻辑：资源错配与扩产周期 
投资者需要关注供给端的两个核心矛盾： 
1.产能周期的错配（导致价格暴涨暴跌的根源）： 上游矿山开发周期极长（5-7年或更久），冶炼厂建设需2-3年 。 下游电池厂扩产极快（仅需6-10个月） 。 结论： 下游需求爆发时，上游产能无法迅速跟进，导致供需错配。 
2.成本差异（成本曲线）： 成本排序： 南美盐湖 < 国内盐湖 < 锂云母 < 锂辉石 。 全球锂资源主要集中在智利（盐湖）、澳大利亚（锂辉石），中国虽然冶炼能力强，但原料高度依赖进口（进口依赖度高） 。 

三、碳酸锂需求端逻辑：双轮驱动（动力+储能） 
主要驱动力： 新能源汽车是核心，储能是新增长极。 动力电池： 磷酸铁锂（LFP）和三元材料是主流。近年来随着技术进步（如刀片电池），LFP占比提升，2022年装车量占比达62% 。 储能电池： 增速极快，2022年同比增长超130%，未来有望成为新的需求支柱 。 
消费区域： 高度集中在电池厂聚集地，前五大消费省份为江苏、湖南、福建、广东、湖北 。 

四、碳酸锂交易节奏：明显的季节性特征 投资者在进行短期交易或库存管理时，需注意以下季节性规律： 
生产淡季（1-2月）： 受青海盐湖结冰、春节检修影响，产量普遍下降30-40% 。 
生产旺季（6月）： 随着气温升高，开工率达到顶峰 。 
夏季扰动（7月后）： 可能受高温限电影响，产量小幅下降 。 

五、影响碳酸锂价格的三大核心维度，
1.供给侧： 原料供应（如澳洲矿山产能、南美船运）、产能利用率（检修/限电）、进口量变化 。 2.需求侧： 政策驱动（双积分、绿牌）正逐步转向市场驱动（技术进步、成本下降） 。 3.成本侧： 采矿及加工成本支撑价格底线 。

碳酸锂核心矛盾分析：极为显著的“时间错配” 这是解释碳酸锂价格暴涨暴跌的最底层逻辑，也是投资者理解周期的关键。 上游“慢” vs 下游“快”： 上游勘探与建设： 新建锂辉石/锂云母冶炼厂需 2-3年，产能爬坡需 1-2年；新建盐湖提锂工厂则长达 5-7年 。 下游扩产： 正极材料企业扩产最短仅需 6-10个月 。 深度解读： 这种 5年以上 vs 1年以内 的巨大时间差，导致需求爆发时，供给端具有极强的刚性（无法短期扩产）。这解释了为何2021-2022年需求爆发时，产能利用率虽有提升，但价格依然飙升 。 投资启示： 关注下游电池厂的扩产公告是“滞后指标”，必须提前3-5年关注上游矿山的资本开支（CAPEX）情况才能预判未来的供需拐点。 
碳酸锂供给侧深度画像：中国特色的“资源-加工”倒挂：
中国在产业链中的特殊地位——加工能力过剩，但资源禀赋不足。 1. 资源结构的“短板” 储量与产量的反差： 中国拥有全球 63% 的碳酸锂产量（加工端），但锂资源储量仅占全球 8% 。 原料高度依赖进口： 2022年进口量创历史新高达13.6万吨，主要来自 智利（盐湖） 和 澳大利亚（锂辉石），两者合计占进口量的 98% 。 深度解读： 中国实际上是全球锂资源的“代工厂”。这种结构使得国内市场极易受海外地缘政治、船运物流（如2022年初的船运受阻）的影响 。 

国内碳酸锂产能的“三足鼎立”与成本分层：
国内生产高度集中在江西（40%）、青海（18%）、四川（17%），代表了三种不同的成本逻辑： 青海（盐湖）： 成本最低，但受限于“老卤提锂”周期长（晒卤需12-24个月）和冬季低温停产 。 四川（锂辉石）： 工艺最成熟，产品质量最稳定，但原料主要依赖澳洲进口，定价权弱 。 江西（锂云母）： 这是中国特有的资源，储量大但品位低、杂质（氟、铷、铯）多，提锂难度高，废渣量大，且硫酸法易腐蚀设备 。 深度解读： 江西锂云母的成本线往往决定了碳酸锂价格的“边际支撑位”。当价格跌破云母提锂成本时，这就不仅是供需问题，而是产能出清的信号。
